# Junmai AutoDev: Lightroom × LLM 自動現像システム

## 概要

**Junmai AutoDev**は、大規模言語モデル（LLM）を活用して、Adobe Lightroom Classicの現像設定を自動化するシステムです。ユーザーが「透明感のあるポートレート」のような自然言語で指示を出すと、システムがそれを解析し、Lightroomの現像パラメータに変換して自動的に適用します。

このプロジェクトは、Lightroomプラグイン（Lua）、ローカルAPIブリッジ（Python/Flask）、そしてローカルLLM（Ollama）を組み合わせて実現されています。

## 特徴

- **自然言語での指示**: 専門的なパラメータ知識がなくても、「こんな感じ」という言葉で写真を編集できます。
- **全自動インストール**: `JunmaiAutoDev_setup.exe` を実行するだけで、必要なコンポーネント（AIエンジン、Lightroomプラグイン等）がすべて自動でセットアップされます。
- **初心者向けUI**: PC初心者の方でも迷わず使える「Guided Flow」モードを搭載。テンプレートを選ぶだけで、プロ品質の編集が可能です。
- **安全な編集**: 編集は常に元の写真の「仮想コピー」に対して行われるため、大切なオリジナルデータが変更されることはありません。

## インストール手順 (Windows)

1.  **インストーラーの実行**:
    リリースページから `JunmaiAutoDev_setup.exe` をダウンロードし、ダブルクリックして実行します。

2.  **画面の指示に従う**:
    インストーラーの指示に従って「次へ」ボタンを押していくだけで、以下のすべてが自動的に行われます。
    - アプリケーション本体のインストール
    - AIエンジン (Ollama) のダウンロードとインストール
    - Lightroom Classic へのプラグインの自動セットアップ

3.  **Lightroom Classic の再起動**:
    インストール完了後、もしLightroom Classicを起動していた場合は、一度再起動してください。

以上で、すべてのセットアップは完了です。

## 使用方法

1.  **Junmai AutoDev を起動**:
    デスクトップに作成された「Junmai AutoDev」ショートカットをダブルクリックして、アプリを起動します。

2.  **雰囲気を指示**:
    「Guided Flow」タブで好みのテンプレートを選ぶか、あなたの言葉で直接、仕上げたい写真の雰囲気を入力します。

3.  **Lightroomへ送信**:
    「Lightroom に送信する」ボタンをクリックします。

4.  **Lightroomで確認**:
    Lightroom Classicで写真を選択していると、数秒後に自動で「仮想コピー」が作成され、AIによる現像設定が適用されます。
