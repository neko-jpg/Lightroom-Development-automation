## 🎯 クイックスタート

### 初めての方

1. **[インストールガイド](INSTALLATION_GUIDE.md)** を読んでシステムをセットアップ
2. **[ユーザーマニュアル](USER_MANUAL.md)** の「基本的なワークフロー」を読む
3. テスト写真で動作確認
4. 本格的に使用開始

### 既にインストール済みの方

- 使い方がわからない → **[ユーザーマニュアル](USER_MANUAL.md)**
- 問題が発生した → **[トラブルシューティング](TROUBLESHOOTING.md)**
- 特定の質問がある → **[FAQ](FAQ.md)**

## 🔍 ドキュメント検索のヒント

### 特定の機能を探す場合

1. **[ユーザーマニュアル](USER_MANUAL.md)** の目次を確認
2. ブラウザの検索機能（Ctrl+F / Cmd+F）を使用
3. 見つからない場合は **[FAQ](FAQ.md)** を検索

### エラーメッセージで検索する場合

1. **[トラブルシューティング](TROUBLESHOOTING.md)** でエラーメッセージを検索
2. 見つからない場合は GitHub Issues で検索
3. それでも解決しない場合は新しい Issue を作成

### 設定方法を探す場合

1. **[ユーザーマニュアル](USER_MANUAL.md)** の「設定のカスタマイズ」を確認
2. `config.json` のサンプルを参照
3. **[FAQ](FAQ.md)** で関連する質問を検索

- **正確性**: 最新のコードと一致する内容を記載
- **構造化**: 見出し、リスト、表を活用して読みやすく
