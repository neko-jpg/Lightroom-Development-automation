# Approval Queue Quick Reference
承認キュー クイックリファレンス

## キーボードショートカット

```
┌─────────────────────────────────────────┐
│  ←  │  前の写真                          │
│  →  │  次の写真                          │
│ Enter│  承認（書き出し待機へ）            │
│ Delete│ 却下                              │
│  M  │  修正                              │
│  S  │  スキップ                          │
└─────────────────────────────────────────┘
```

## ボタン

```
✓ Approve  - 承認して次へ
✗ Reject   - 却下して次へ
✏️ Modify   - パラメータ修正
⏭️ Skip     - スキップして次へ
```

## 画面レイアウト

```
┌──────────────┬──────────────┬──────────┐
│   Before     │    After     │ AI Score │
│   (元画像)   │  (現像後)    │  ★★★★☆  │
│              │              │          │
│              │              │ Details  │
│              │              │ Context  │
│              │              │ Preset   │
│              │              │ Params   │
└──────────────┴──────────────┴──────────┘
         [Approve] [Reject] [Modify] [Skip]
```

## ワークフロー

```
1. 写真確認
   ↓
2. 判断
   ├─ 良い → Enter
   ├─ 悪い → Delete
   └─ 保留 → S
   ↓
3. 次の写真へ自動移動
```

## AI評価スコア

- **Overall**: 総合評価（1-5星）
- **Focus**: ピント精度
- **Exposure**: 露出適正
- **Composition**: 構図バランス

## Tips

💡 **高速処理**: キーボードのみで操作
💡 **スキップ活用**: 迷ったら後回し
💡 **進捗確認**: 画面上部で残り枚数を確認
