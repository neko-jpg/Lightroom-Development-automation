## ウィジェット詳細

### 1. System Status Widget
**位置**: 上部
**高さ**: 固定（約100px）
**更新**: 5秒ごと

**表示項目**:
- システム稼働状態（Running/Stopped）
- LLMモデル情報（Ollama + モデル名）
- Lightroom接続状態（Connected/Disconnected）
- GPU状態（温度、メモリ使用量）
- 今日の統計（処理枚数、成功率、平均時間）

**色分け**:
- 緑（●）: 正常
- 黄（●）: 警告
- 赤（●）: エラー
- 灰（●）: 不明/無効

### 2. Active Sessions Widget
**位置**: 中央上部
**高さ**: 可変（スクロール可能）
**更新**: 3秒ごと

**表示項目**:
- セッション名（フォルダアイコン付き）
- 進捗バー（処理済み/総枚数）
- ステータス（importing/selecting/developing/exporting/completed）
- ETA（推定残り時間）

**インタラクション**:
- クリックでセッション詳細画面へ遷移
- ホバーでカーソルが指マークに変化

### 3. Quick Actions Widget
**位置**: 中央
**高さ**: 固定（約80px）
**更新**: 5秒ごと（承認キュー数のみ）

**ボタン**:
- 📂 Add Hot Folder - ホットフォルダー追加ダイアログ
- ⚙️ Settings - 設定タブへ遷移
- 📊 Statistics - 統計画面表示
- ✅ Approval Queue (数) - 承認タブへ遷移
- 📤 Export Now - 即座に書き出し実行

### 4. Recent Activity Widget
**位置**: 下部
**高さ**: 可変（スクロール可能）
**更新**: 2秒ごと

**表示項目**:
- タイムスタンプ（HH:MM形式）
- ログレベルアイコン（✓/⚠/✗/●）
- メッセージ内容

**色分け**:
- 緑: INFO（成功）
- 黄: WARNING（警告）
- 赤: ERROR（エラー）
- 青: DEBUG（デバッグ）

## レスポンシブ動作

### ウィンドウサイズ変更時
- System Status: 高さ固定、幅は親に追従
- Active Sessions: 高さ可変（最小100px）、スクロール表示
- Quick Actions: 高さ固定、ボタンは2行グリッド配置
- Recent Activity: 高さ可変（最小100px）、スクロール表示

### 最小ウィンドウサイズ
- 幅: 800px
- 高さ: 600px

## テーマ対応

### ダークテーマ（デフォルト）
- 背景: #1e1e1e
- セッションカード: #2b2b2b
- テキスト: #ffffff
- 境界線: #3c3c3c

### ライトテーマ
- 背景: #ffffff
- セッションカード: #f5f5f5
- テキスト: #000000
- 境界線: #e0e0e0

## アクセシビリティ

- すべてのボタンにキーボードショートカット対応
- 色だけでなくアイコンでも状態を表現
- 十分なコントラスト比（WCAG AA準拠）
- スクリーンリーダー対応（aria-label設定）

## パフォーマンス考慮

- 各ウィジェットは独立したタイマーで更新
- API呼び出しは非同期（UIブロックなし）
- タイムアウト: 2秒
- エラー時は前回の表示を維持
- ウィンドウ非表示時は更新を一時停止（将来実装）
